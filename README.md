# krul-cli

## Usage

```js
$ npm i krul-cli -g  or yarn global add krul-cli
$ krul-cli
```

## Support template list

- next-app (next.js app)
- react-library (js & ts)
- node-cli-library (todo)
- typescript-library (todo)
- electron app (todo)
